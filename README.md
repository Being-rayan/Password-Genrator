**Password Generator**

A simple Java console application that generates secure and customizable passwords based on user input. The user specifies the total password length and the number of special characters, and the program creates a randomized password accordingly.

This project demonstrates core Java concepts such as input handling, random number generation, string manipulation, validation logic, and character shuffling to ensure password randomness.

How to Run

Compile the program:
javac PasswordGenerator.java


Run the program:

java PasswordGenerator

Enter the required password length and number of special characters when prompted.
